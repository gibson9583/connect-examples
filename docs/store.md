# Connect Examples

Example **channels** and **code templates** for Open Integration Engine, from the
NextGen `connect-examples` repository — installable directly through the Community
Store. Every example in the repo is listed individually.

## What's inside

**Channels**
- **Get Num Pages From Embedded PDF** — count the pages of a PDF embedded in a message.
- **Validate XSD Source Filter** — validate a message against an XSD in a source filter.

**Code templates** (16) — HL7 helpers, XML utilities, PDF tools, and more, including:
Convert XML to HL7, Fix HL7 Node Order, Rename HL7 Field, Join HL7 Subcomponents,
Strip Empty Nodes from XML, Encrypt PDF, Extract Text From PDF, Get Num Pages In PDF,
Thread-safe get-or-create from globalMap, and others. Each installs individually into
a library you choose.

**Code template library**
- **Route with Sources** — a ready-made library.

## How install works

Channels and code templates are **imported** through the engine (not installed as
extensions), so they take effect immediately — no engine restart:

- A **channel** appears in the channel list.
- A **code template library** is added to your libraries with all its templates.
- A **code template** prompts you to add it to a **new** library or an **existing** one.

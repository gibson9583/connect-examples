# Connect Examples

A collection of example **channels** and **code templates** for Open Integration
Engine, from the NextGen `connect-examples` repository — installable directly through
the Community Store.

## What's inside

**Channels**
- **Get Num Pages From Embedded PDF** — count the pages of a PDF embedded in a message.
- **Validate XSD Source Filter** — validate a message against an XSD in a source filter.

**Code Templates**
- **Connect Examples** (library) — 16 utility code templates: HL7 helpers, XML
  utilities, PDF tools, date conversion, and more, grouped into one library.
- **Convert XML to HL7** and **Encrypt PDF** — individual code templates you can add
  to any library.

## How install works

Channels and code templates are **imported** through the engine (not installed as
extensions), so they take effect immediately — no engine restart:

- A **channel** appears in the channel list.
- A **code template library** is added to your libraries with all its templates.
- A **code template** prompts you to add it to a **new** library or an **existing**
  one.
